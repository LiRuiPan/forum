# Forum
**1 简介**

一个可实用的论坛项目,基于flask框架、jinja模板、sqlalchemy ORM框架、MySQL和redis数据库。


**2 功能实现**

- 实现用户找回密码功能
- 实现用户注册、登陆、登出功能
- 实现用户个人主页展示相关动态功能
- 实现用户更改个人信息、密码、头像功能
- 实现查找用户、帖子功能
- 实现对帖子发表、删除评论功能
- 实现会话状态记录、权限判断功能
- 实现在对应版块发表、删除帖子功能
- 实现站内发出、回复、删除私信功能
